# Checklist de Entrega — Sistema Hotelero Central Beach

Usa esta lista para confirmar que el sistema quedó correctamente instalado y listo para operar, antes de entregarlo al cliente.

## Instalación técnica

- [ ] Google Sheet creada
- [ ] `setupSheets()` ejecutado sin errores
- [ ] Apps Script publicado como aplicación web
- [ ] `API_BASE_URL` configurado en `app.js` y API conectada (aparece "Google Sheets activo")

## Datos reales del hostal

- [ ] Habitaciones reales creadas (no solo las de ejemplo)
- [ ] Al menos un huésped real registrado

## Pruebas funcionales

- [ ] Reservación de prueba creada
- [ ] Check-In probado
- [ ] Check-Out probado
- [ ] Pago probado (y el saldo se actualizó correctamente)
- [ ] Reporte de cierre probado (se generó e imprimió correctamente)
- [ ] Comprobante probado (se generó e imprimió correctamente)
- [ ] WhatsApp probado (el mensaje se abrió correctamente con los datos de la reservación)

## Respaldo y capacitación

- [ ] Backup verificado (se descargó o se confirmó el historial de versiones de Google Sheets)
- [ ] Cliente capacitado en el uso diario del sistema (Dashboard, Habitaciones, Huéspedes, Reservaciones, Calendario, Check-In, Check-Out, Pagos, Reportes, Comprobante y WhatsApp)

## Documentación entregada

- [ ] `README.md` entregado
- [ ] `MANUAL_USUARIO.md` entregado al equipo de recepción
- [ ] `INSTALACION.md` entregado a quien administre el sistema
- [ ] Enlace al sistema (URL final donde queda publicado `index.html`) entregado al cliente
